# Multi-Skill Collection

This repository contains a standardized collection of AI agent skills.

## Structure
- `skills/`: The standardized directory containing all available skills.

## How to Install (Multica)
Paste the direct URL to the skill directory you want to import.

**Format:**
`https://github.com/ckccbaize/skill-/tree/main/skills/<skill-name>`

**Example:**
`https://github.com/ckccbaize/skill-/tree/main/skills/brainstorming`

## Available Skills in `skills/`
- `brainstorming`
- `systematic-debugging`
- `test-driven-development`
- `using-superpowers`
- ... (see the `skills/` directory for the full list)
